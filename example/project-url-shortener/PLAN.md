## 风险与缓解

| 风险 | 可能性 | 影响 | 缓解措施 |
|------|--------|------|---------|
| Redis 缓存击穿（热点短码大量并发） | 中 | 高 | 缓存加锁 + 布隆过滤器预热 |
| nanoid 碰撞（8 位短码重复） | 低 | 中 | 生成时检查 DB 唯一性，碰撞重试最多 3 次 |
| PostgreSQL 连接池耗尽 | 低 | 高 | 配置 asyncpg 连接池上限 + 监控告警 |
