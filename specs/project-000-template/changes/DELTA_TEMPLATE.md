```markdown
### REQ-020: 手机号注册

- **描述**：用户可通过手机号 + 短信验证码注册账号
- **验收标准**：
  - 发送验证码接口：POST /api/v1/auth/sms/send，60 秒内不可重发
  - 注册成功后，手机号全局唯一
  - 无效手机号格式返回 HTTP 422
- **优先级**：P1
- **状态**：draft
- **关联**：REQ-001（与邮箱注册并列，共用 user 表）
```
